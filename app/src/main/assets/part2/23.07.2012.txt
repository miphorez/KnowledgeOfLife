Сила духа человека духа будет зависеть от энергии.
Энергия может быть почерпнута только из творчества.
Творчество будет вести человека духа по пути эволюции духа.
Только творчество может дать необходимую энергию.
Энергия творчества поднимет человека.
Творчество будет касаться всех областей жизни человека духа.
Творчеством будет пропитана вся жизнь человека духа.
Не будет в жизни места вне творчества.
Творчество будет править всей жизнью.
Не будет жизни вне творчества.
